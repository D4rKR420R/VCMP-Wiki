### Message
```Squirrel
Message(message)
```

Sends a message to all players in server.

#### Arguments
- [string] *message* - The message to be sent.

#### Returns
> void

#### Examples
```Squirrel
Message("Have fun!");
```

> Hexadecimal color codes can be used to color the message string, enclosed with square brackets.

```Squirrel
Message("[#0000FF]Have fun!");
```
